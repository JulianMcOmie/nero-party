# Project Guidelines for Claude

## CRITICAL: Never Auto-Commit or Auto-Push

**NEVER automatically commit or push changes.** You must wait for explicit instruction from the user before committing or pushing. This is non-negotiable.

- Do NOT use `git commit` without the user asking
- Do NOT use `git push` without the user asking
- Always wait for the user to review changes and ask you to commit

This applies even when fixing bugs or making changes the user requested. Always ask first or wait for explicit instruction.
